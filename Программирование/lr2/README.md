# Лабораторная работа 2

## Микросервисы для конвертации валюты и анализа истории курса валюты

### Сборка и запуск

Соберите Docker-образ:
```bash
docker build -t currency_converter .
```

Запустите контейнер:
```bash
docker run -p 5000:5000 -p 5001:5001 currency-converter
```
